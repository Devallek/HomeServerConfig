# Services Docker pour héberger un serveur Mail

## Container et images utilisés

- docker-mailserver:latest

## Port mapping (Sortant)

- 25 : SMTP
- 143 : IMAP
- 465 : SMTPS
- 587 : SMTP Submission
- 993 : IMAPS
